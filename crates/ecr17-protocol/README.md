# ecr17-protocol

Italian **ECR17** payment protocol (Nexi Group POS terminals) over LAN — a pure-Rust
protocol engine plus an async client and an optional tokio TCP transport.

> This is the crates.io front page. The full documentation, screenshots and the Tauri
> control-panel demo live in the repository README:
> <https://github.com/padosoft/rust-ecr17-protocol>

Part of the ECR17 family: [React Native](https://github.com/padosoft/react-native-ecr17-protocol)
· [Laravel](https://github.com/padosoft/laravel-ecr17) · **Rust/Tauri (this crate)**.
